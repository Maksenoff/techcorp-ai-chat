# Lien Google Colab

Après le push sur la branche `groupe-ia-13` du dépôt officiel, ouvrir le notebook avec ce lien :

[Ouvrir le notebook médical dans Google Colab](https://colab.research.google.com/github/H04K/hackathon_ynov/blob/groupe-ia-13/rendu/ia/notebook_finetuning_medical.ipynb)

Si la branche n'est pas encore poussée, aller sur [Google Colab](https://colab.research.google.com/), choisir **Fichier > Importer un notebook**, puis sélectionner `notebook_finetuning_medical.ipynb`.

Avant l'exécution :

1. sélectionner **Exécution > Modifier le type d'exécution** ;
2. choisir un GPU T4 ;
3. exécuter toutes les cellules ;
4. télécharger `medical-lora-groupe-13.zip` ;
5. reporter les métriques dans `RESULTATS_FINETUNING.md`.
